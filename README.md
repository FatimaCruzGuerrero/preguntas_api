# Trivia de Preguntas — Proyecto con Gemini API

## Descripción
Este proyecto es una **aplicación web interactiva** que genera preguntas de opción múltiple sobre temas de **JavaScript, HTML y CSS** utilizando la **API de Gemini** (Google AI).  
Cada pregunta incluye cuatro opciones, una respuesta correcta y una breve explicación.  
El sistema lleva un conteo de respuestas **correctas e incorrectas**, que se guardan localmente en el navegador mediante `localStorage`.

---

## Funcionalidades principales
- Generación automática de preguntas mediante la API de Gemini.  
- Interfaz moderna y responsiva gracias a **Bootstrap 5**.  
- Conteo persistente de aciertos y errores.  
- Resaltado visual de respuestas correctas e incorrectas.  
- Explicación inmediata después de responder.  

---

## Estructura del proyecto
El proyecto está organizado de la siguiente manera:
- `preguntas.html`: Interfaz principal
- `script.js`: Lógica y conexión con Gemini
- `README.md`: Documentación del proyecto

---

## Tecnologías utilizadas
- **HTML5**  
- **CSS3 (Bootstrap 5.3)**  
- **JavaScript (fetch API, JSON, localStorage)**  
- **Gemini API (modelo: `gemini-2.5-flash`)**
